API Reference v1.0

Documentation for the Nexus Portal API for developers and integrators.

GET /api/status
Check system health status.
POST /report
Submit a support ticket or URL for admin review.
Parameters: url (string, required)
POST /make_admin Internal
Promote a user to Administrator role.
Parameters: username (string, required)

Note: This endpoint is restricted to Administrator use only. Accessible via POST or GET for legacy support.

GET /admin/logs Admin
Retrieve system logs.
Parameters: logfile (string, optional)
